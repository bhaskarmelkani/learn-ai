Once we accept that one score is not always enough, the next step is natural:

let the model learn several internal detectors, then combine them.

That system is called a **neural network**.

## One Neuron

A neuron is still doing a familiar job:

1. compute a score
2. apply a non-linearity
3. pass the result forward

So a neuron is not magic. It is a small detector.

## Several Neurons in Parallel

When we place many neurons in a layer, each one can specialize in a different pattern.

One neuron might respond to one feature. Another might respond to a different one. The next layer can combine them into richer patterns.

## The Big Intuition

Neural networks matter because they can build internal representations automatically instead of forcing us to hand-design every useful feature ourselves.
