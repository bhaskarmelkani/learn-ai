Drawing a good line by hand is helpful for intuition.

But once we have many houses, we want the model to improve its own line from data.

That requires two ideas:

1. a way to measure how wrong the line is
2. a way to update the line so it becomes less wrong

## Measuring Error

For one example, the error is simple:

$$\text{error} = \text{actual} - \text{prediction}$$

For many examples, we need one summary number. A common choice is **mean squared error (MSE)**:

$$\text{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

The name matters less than the intuition: lower MSE means the line fits the data better.

## Improving the Line

Once we can measure badness, we can ask a new question:

how should we change the line so that badness goes down?

That update process is called **gradient descent**.

## One Weird Point Can Move the Whole Story

Not all data points have the same effect on the fitted line.

If one house sale looks very unusual, it can pull the line away from the main cluster.
