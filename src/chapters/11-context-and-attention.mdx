For language, context matters.

The token after "The cat sat on the" should be influenced by what came before it.

That need leads to **attention**: a way for tokens to weigh which other tokens matter for the next prediction.

## Attention as Context Selection

At a high level, attention asks:

- which earlier tokens matter most right now?
- how strongly should they influence this position?
- how should that context change the next prediction?

This does **not** mean the model "understands" context like a person. It means the architecture has a learned way to move information between tokens.

## Predict the Next Token

Once the model has context-aware representations, it produces a probability distribution over possible next tokens.

That distribution is the immediate output of the language model.

## Why the Transformer Matters

The architecture built around attention is called the **transformer**.

Transformers matter because they let tokens interact with one another in a flexible, learned way instead of treating every token in isolation.
